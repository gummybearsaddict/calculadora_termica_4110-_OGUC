# calculadora_termica_4110-_OGUC
Calculadora de Cumplimiento Térmico (Art. 4.1.10 OGUC), permite verificar el cumplimiento de la Actualización de la Reglamentación Térmica (D.S. N°15 MINVU), vigente desde Noviembre 2025. Utiliza los criterios de Zonificación y Transmitancia Térmica de los documentos DITEC.
